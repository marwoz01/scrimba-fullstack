# Retro Rides 

Projekt polegał na stworzeniu bazy danych dla salonu z klasycznymi samochodami **Retro Rides**, zarządzanej przez CEO Rodney’a przy użyciu SQL.

Podczas modułu przećwiczyłem:
- tworzenie zapytań z `SELECT`, `FROM` i `WHERE`
- filtrowanie danych za pomocą warunków (`AND`, `OR`, `BETWEEN`, `IN`, `LIKE`)
- sortowanie wyników (`ORDER BY`)
- agregowanie danych (`COUNT`, `AVG`, `MIN`, `MAX`, `GROUP BY`, `HAVING`)
- modyfikowanie danych (`INSERT`, `UPDATE`, `DELETE`)
