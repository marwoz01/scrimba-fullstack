# Databases 

Rozdział **Databases** z kursu Fullstack Developer na Scrimba.

Celem było poznanie podstaw pracy z bazami danych oraz nauka pisania zapytań SQL od podstaw.  
Podczas rozdziału zrealizowałem projekty i ćwiczenia obejmujące:

- **Intro to Databases** – wprowadzenie do działania baz danych i ich zastosowania w aplikacjach
- **Writing SQL Queries** – tworzenie zapytań przy użyciu `SELECT`, `WHERE`, `ORDER BY`, `GROUP BY`, `HAVING` oraz funkcji agregujących
- **Creating and Joining Tables** – tworzenie tabel, definiowanie relacji między nimi i łączenie danych przy użyciu `JOIN`

Dzięki temu rozdziałowi nauczyłem się skutecznie pobierać, filtrować, sortować i modyfikować dane w SQL oraz przygotowywać struktury baz danych pod aplikacje webowe.
