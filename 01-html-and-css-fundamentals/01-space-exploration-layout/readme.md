# Space Exploration – HTML & CSS Fundamentals

W tym projekcie przećwiczono podstawy stylizacji i struktury strony:

- dodawanie **obrazów tła** i wykorzystanie formatu **WebP**
- włączenie **czcionek Google Fonts**
- użycie znacznika **`<span>`** do wyróżnień w tekście
- stosowanie **identyfikatorów i klas użytkowych**
- dodanie efektu **text-shadow** dla nagłówków

Projekt stanowi podsumowanie kluczowych elementów HTML i CSS.
