# HTML & CSS Fundamentals

Sekcja poświęcona podstawom tworzenia stron internetowych.  
Obejmuje naukę struktury HTML i stylowania w CSS - od znaczników i klas, po pozycjonowanie elementów, kolory, czcionki i efekty wizualne.  
Projekty w tej części pozwalają zrozumieć, jak łączyć treść z wyglądem, tworząc estetyczne i responsywne strony.
