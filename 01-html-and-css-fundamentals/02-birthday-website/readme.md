# Birthday Website – HTML & CSS Fundamentals

W tym projekcie przećwiczono kluczowe techniki układu i stylizacji w CSS:

- użycie właściwości **`align-items`** i **`flex-direction`** do pozycjonowania elementów w Flexboxie
- zastosowanie pseudoklasy **`:hover`** do interaktywnych efektów
- tworzenie tła z wykorzystaniem **gradientów**
- użycie **grouping selector** do wspólnego stylowania wielu elementów

Projekt stanowi praktyczne ćwiczenie z Flexboxa i zaawansowanych selektorów CSS.
